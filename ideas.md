# Career Portal Design Ideas

## Approach 1 — Institutional Precision (Chosen)
<response>
<text>
**Design Movement:** Institutional / Government-grade Information Architecture
**Core Principles:**
- Functional clarity over decoration — every element earns its place
- Dense information hierarchy with clear visual weight
- Navy/indigo sidebar authority paired with clean white content area
- Structured tabular data presentation

**Color Philosophy:** Deep navy (#1e2a5e) sidebar conveys institutional trust and authority. White content area ensures readability. Teal/blue accent badges for counts. Purple active-state highlights match university branding.

**Layout Paradigm:** Fixed left sidebar (280px) + fixed top navigation bar + scrollable main content. Classic enterprise dashboard layout with collapsible sidebar sections.

**Signature Elements:**
- Circular teal badge counters on quick-search items
- Purple active sidebar highlight
- Horizontal top nav with arrow scroll controls

**Interaction Philosophy:** Accordion sidebar sections expand/collapse. Hover states on nav items. Active state clearly highlighted.

**Animation:** Subtle accordion expand/collapse. Hover transitions on sidebar items (150ms ease).

**Typography System:** System sans-serif (Segoe UI / Arial) for institutional feel. Small caps section headers. Bold user name in sidebar.
</text>
